# push-cli

Ajoute la commande `push` en CLI pour Git.

![Description de l’image](./SINJ.jpg)

## Utilisation

```bash
push "message de commit"
```

Cela fait :

1. `git add .`
2. `git commit -m "message de commit"`
3. `git push`

## Installation

```bash
npm install -g push-cli
```

## Prérequis

- Node.js
- git installé
